- ### 폭발한계
	- 폭발한계 = 연소한계 = 연소범위 = 가연한계 = 가연범위
	- 물질의 연소범위
		- | 가스         | 하한계(vol%) | 상한계(vol%) |
		  | ------------ | ------ | ------ |
		  | 아세틸렌     | 2.5    | 81     |
		  | 수소         | ==4==  | ==75== |
		  | 일산화탄소   | 12.5   | 74     |
		  | 프로판       | ==2.1==    | ==9.5==       |
		  | 디에틸에테르 | 1.9    | 48     |
		  | 부탄         | 1.8    | 8.4    |
		  | 이황화탄소   | 1.2    | 44     |
		  | 메탄         | 5      | 15     |
	- 혼합가스의 폭발하한계 (vol%)
		- $\dfrac{100}{L} = \dfrac{V1}{L1}+\dfrac{V2}{L2}+\dfrac{V3}{L3}$
			- L:혼합가스의 폭발하한계
			- L1 : 가연성 가스의 폭발하한계
			- V1 : 가연성 가스의 용량
	- 연소범위
		- 아세틸렌 > ==수소== > ...

- ### 온도
	- #### 인화점
		- 연료가 혼합기를 이룬상태에서 불꽃에 의해 불이 붙는 온도
		- 암기 : ==인산이아에==  (낮은순서)
		- 프로필렌 (-107) < ==산화프로필렌==(-37) < 이황화탄소(-30) < 아세톤(-18) < 에틸알콜(13) < 등유(43~72) < 경유(50~70)
	- #### 착화점(발화점)
		- 불꽃 없이도 온도를 올리다보면 공기중의 산소와 연료가 산화를 시작하면서 자연히 발화하는 온도
		- 암기 : 착황이 (낮은순서)
		- ==황린==(30~50) < 황화린,==이황화탄소==(100) < 가솔린 < 아세톤 < 벤젠
	- #### 연소점
		- 연료에 불씨를 갖다 댈 때 계속해서 연소 할 수 있는 온도
	- #### 연소의 색과 온도

- ### 물과 화학반응
	- 주수소화시 가연물에 따라 발생하는 가연성 가스의 종류 : 탄칼아 / 탄알메 / 인포 / 수수
	- 탄화칼슘 + 물 -> 수산화칼슘 + ==아세틸렌==
	  	$CaC_2 + 2H_2O -> Ca(OH)_2 + C_2H_2$
	- 탄화알루미늄 + 물 -> 수산화알루미늄 + ==메탄==
	- 인화칼슘 + 물 -> 수산화칼슘 + ==포스핀==
	- 수산화리튬 + 물 -> 수산화리튬 + 수소
- ### 물의 소화능력
	- 다른물질보다 비열이 크다
	- 다른물질보다 증발잠열(기화잠열)이 크다
	- 다른물질보다 ==융해잠열이 크다==
	- 주된 소화효과 : 냉각효과
	- 무상주수시 질식소화 가능

	- 단위 : $W/m\cdot K$
- ### 이상기체 상태방정식
	- $PV=nRT=\dfrac{W}{M}RT$
		- P : atm
		- V : $m^3$
		- n=몰수
		- W=무게 (kg)
		- R : 기체상수 $0.082atm\cdot m^3/kmol \cdot K$
		- M : 기체의 분자량 (이산화탄소 = 44kg/kmol)
		- V가 리터단위이면 W는 g 단위이다 (세제곱미터이면 kg)
- ### 보일 샤를의 법칙
