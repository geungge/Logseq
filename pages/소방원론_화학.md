- ### 화학 계산문제 정리
  collapsed:: true
	- #### 44g 프로판 1몰이 완전연소할 경우 발생한 이산화탄소 부피는 ?
		- 완전연소 : 기체 + O2 -> 이산화탄소 + H2O 의 형태의 분자식을 만든다
		  부피 : 표준상태에서 1mol 기체는 0도 기압에서 ==22.4L== 가짐
		  [@기출](((63f38fb4-378a-4a52-9526-e8294f668836)))
	- #### 수소 1kg이 완전연소할 때 필요한 산소량
	  collapsed:: true
		- 2H2 + O2 -> 2H2O
		  2H2 : 4kg / kmol , O2 : 32 kg/kmol
		  4:32 = 1:x , x = 8kg
		  [기출](((63f394f6-b1eb-4dc9-b849-d96aa793905a)))
	- #### 완전연소 시키기 위해 필요한 산소의 mol 수는?
	  collapsed:: true
		- 1) 기체 + O2 -> 이산화탄소 + H2O 의 형태의 분자식을 만든다
		  2) 이때 산소(O2)의 몰수가 정답이다
	- #### 가연성기체 1몰 완전연소에 필요한 이론 공기량 ?
	  collapsed:: true
		- 필요한 이론 공기량 = 필요 산소 몰수 / 공기 중 산소농도 
		  아세틸렌(C2H2) = 11.9몰 (=2.5/0.21)
		  [기출](((63f39820-5d42-4bc0-8f22-49f0b6fa88c6)))
	- #### MOC(최소산소농도)
	  collapsed:: true
		- ==moc = 산소의 몰수 x 기체의 연소범위 하한값==
		  ==가장 작은 값 : 메탄== (메탄<에탄<프로판<부탄)
		  [기출](((63f399cd-87b5-446f-aa1b-e5a5eb72ae99)))
	- #### 중량백분율이 주어지고 화합물의 분자식을 구하라
	  collapsed:: true
		- 1) 각 원소의 몰수의 비율을 구한다 ( 중량백분율 / 원자량 )
		  2) 구해진 몰수의 비율로 분자식을 구한다
		  [기출](((63f39a6b-4476-404d-9b84-8d7245bfc0eb)))
	- #### 이산화탄소약재 방출시 혼합공기의 분자량
	  collapsed:: true
		- 1) CO2 방출 후 산소농도를 대입하여 방출 후 CO2 농도 계산
		  2) 산소,이산화탄소 농도를 알았으면 질소의 농도를 구할 수 있다
		       질소의 농도 = 100 - (산소농도 + 이산화탄소 농도)
		  3) 각 원소 분자량 x 농도 (vol%)
		  [기출](((63f39b89-16fc-4e25-a2d7-30268edfe048)))
	- #### 이산화탄소 증기비중
	  collapsed:: true
		- ==증기비중 = 분자량 / 29==   (29 = 공기의 평균분자량)
		  이산화탄소 증기비중 = 44/29 = 1.52
	- #### 이산화탄소 20g은 몇 mol인가?
	  collapsed:: true
		- 이산화탄소 분자량 계산 결과 = g/mol
		  비례식으로 풀면된다.
		  [기출](((63f39dea-55a0-4c1f-bf8b-3cf849d75007)))
	- #### 이산화탄소 농도
	  collapsed:: true
		- 이산화탄소 농도 = (21-산소농도) / 21
		- 이산화탄소 농도를 구하라
		- 한계 산소 농도를  구하라
	- #### 몰분율
	  collapsed:: true
		- 몰분율 = 어떤 성분의 몰수 / 전체 몰수 ( 몰수 = 질량/분자량)
		  [기출](((63f39f91-2de0-4671-9349-7be622d6aa3f)))
	- #### 평균분자량
	  collapsed:: true
		- 공기의 평균분자량 = 질소 중량 퍼센트 x28 + 산소 중량퍼센트 x 32 = 28x0.792+32x0.208 = 28.83
		  [기출](((63f39ee4-86e1-4b94-a399-d990eac663bb)))
	- #### 이상기체상태방정식
	  collapsed:: true
		- $PV=nRT=\dfrac{W}{M}RT$
		- 주의사항 (R=0.082 , V:세제곱미터 , W:kg)
		- 기체의 분자량 (부피,질량,기압,온도가 주어진다)을 구하라
		- 이산화탄소를 액화하여 얻을 수 있는 액화탄산가스의 무게를 구하라
	- #### 보일-샤를의 법칙
	  collapsed:: true
		- 온도 변화에 따른 압력은 어떻게 변하는가?
		- 기체가 차지하는 부피는 압력에 반비례, 온도에 비례한다
		- 이상기체에서 부피는 일정하다
		- $\dfrac{P_1V_1}{T_1} = \dfrac{P_2V_2}{T_2}$
	- #### 확산속도
	  collapsed:: true
		- 루트 (기체의 분자량 / 공기의 분자량)
		  [기출](((63f3a08f-7934-4762-ba00-9f406d2a703f)))
	- #### 혼합가스의 폭발하한계
	- #### 메탄의 밀도는?
	  collapsed:: true
		- CH4 
		  분자량 16
		  밀도 = 분자량 / 22.4 = 16/22.4 = 0.71 g/l
		  [기출](((63f3a10d-ef4f-4569-a1f9-c469b3fbac85)))
	- #### 1기압,0도,1세제곱미터에 할론1301 0.32kg이 방사되었다. 농도는?
	  collapsed:: true
		- 할론1301 분자량 = 149
		  이상기체상태방정식으로 체적을 구한다
		  농도 = 할론 체적 / (1세제곱미터 + 할론 체적)
		  [기출](((63f3a24b-bc30-471a-9e30-7a8ccf268cb6)))
	- #### CO2 분자량,몰수
	  collapsed:: true
		- $CO_2$ 분자량 = 12+16x2 = 44g/mol
		- $CO_2$ 20g의 몰수 = 0.45mol

### 화학식
collapsed:: true
	- #### 연소 반응
		- 메탄 : $CH_4 + 2O_2 -> CO_2 + 2H_2O$
		- 부탄 : $2C_4H_{10} + 13O_2 -> 8CO_2 + 10H_2O$
		- 수소 : $2H_2 + O_2 -> 2H_2O$
		- 아세틸렌 : $2C_2H_2+5O_2 -> 4CO_2 +2H_2O$
	- #### 물질의 화학식
		- 메에프부
			- 메탄 : CH4 (하한계 5%)
			  에탄 : C2H6 (하한계 3%)
			  프로판 : C3H8 (하한계 2.2%)
			  부탄 : C4H10 (하한계 1.8%)
			  아세틸렌 : C2H2
		- 인산암모늄 : $NH_4H_2PO_4$
		- 암모니아 : NH3
		- HFC-125 : $CHF_2CF_3$
		- 산화칼슘 : CaO
		- 탄산칼슘 : CaCO3
		- 탄화칼슘 : CaC2
